# Resolvable Web
Resolvable Rating &amp; Comment

## Installation
Run Terminal, install node modules
```sh
npm i
```

Run the solution
```sh
npm run dev
```
## Tech
- Vite - Runtime (https://vitejs.dev/)
- Dummy API - (https://dummyapi.io/explorer)
